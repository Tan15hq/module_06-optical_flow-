# Optical Flow and Motion Tracking

This project analyzes motion in two videos using optical flow.

Videos used:
1. puppy.mp4
2. blind_man.mp4

Implementation:
Python + OpenCV (Google Colab)

Steps to run:

1. Open optical_flow_analysis.ipynb
2. Upload videos
3. Run all cells
4. Outputs will be generated in the outputs folder

Outputs generated:

- puppy_optical_flow.mp4
- blind_man_optical_flow.mp4
- puppy_tracking.png
- blindman_tracking.png
